Antes de usar o editar el SISVEN, realizar los siguientes pasos:

	Instalar el SQL Server (versiones 2007 en adelante).

	Una vez instalado el SQL Server, desplegar el servidor, click derecho en la carpeta "Bases de datos->Restaurar base e datos..." y 
	utilizar el archivo Tienda.bak para la restauracion.


Pasos para utilizar SISVEN, sofware hecho en c++ con SQL Server 2014:

	1)Entrar a la Carpeta Proyecto

	2)Entrar a la Carpeta Debug

	3)Ejecutar el archivo Proyecto.exe

	NOTA: Verificar los usuarios en la tabla Usuarios de la base de datos Tienda para poder acceder al sistema.
	Los usuarios con estado 0 podran acceder al sistema.


Pasos para ver y/o editar el Código del SISVEN:

	1)Tener instalado el programa Microsoft Visual Studio (versiones 2010 en adelante).

	2)Abrir el archivo Proyecto.sln, haciendole doble click o usando la funcion "Nuevo->Abrir->Proyecto o solucion..." desde el Visual Studio.
	
	3) Ver los componentes del Proyecto a través del "Explorador de Soluciones".

	4) OPCIONAL - Presionar F5 en el Visual Studio para correr el SISVEN.
